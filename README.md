# Fraud Detection Dashboard

## 📌 Project Overview
This project analyzes transaction data to detect fraud patterns using Power BI and SQL.

## 📊 Features
- Fraud vs Non-Fraud analysis
- Category-wise fraud detection
- State-wise filtering
- Time-based fraud trends

## 🛠️ Tools Used
- Power BI
- SQL
- Excel / Dataset

## 📁 Files Included
- .pbix file (Power BI dashboard)
- .sql file (dataset)
- images (dashboard screenshots)
